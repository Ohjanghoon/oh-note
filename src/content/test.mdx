#

### React의 메모이제이션 (Memoization)

#

React에서 메모이제이션(Memoization)은 불필요한 연산을 방지하고 성능을 최적화하는 중요한 개념입니다.<br/>
렌더링 시 매번 동일한 계산을 반복하는 것은 비효율적이므로, React에서는 `useMemo`와 `useCallback`을 제공하여 이를 최적화할 수 있습니다.

이 글에서는 메모이제이션이 무엇인지, 언제 활용하면 좋은지, 그리고 실전 예제까지 다뤄보겠습니다.

# `useMemo` - 연산된 값(Value) 메모이제이션
